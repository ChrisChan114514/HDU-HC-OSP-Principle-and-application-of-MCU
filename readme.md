## 实验四的更新

这是数码管及按键应用实验，数码管的显示放在了while(1)的主线程，没有进中断（没啥必要），

但是秒表实现定时器之类还是要中断（这也刚好错开，不然两个定时器溢出中断会有点抽象）。
